When Towers Fall, LoRa Stands

A pocket-size mesh network that lets every responder keep using the familiar Rescue 239 app — minus the cellular dependency.

Step-by-Step Workflow

  1. Grab & Go Box. Two pre-assigned unit members store a shoebox with 8-12 LoRa nodes at home.
  2. Rapid Station Setup. On arrival, they place a central node on a tripod, power it with any USB-C power bank, and flip it on.
  3. Bluetooth Pairing. Responders pass the station, tap “Pair” in the app; phone ↔ node handshake takes <1 min.
  4. Mesh Online. Every report now travels over LoRa mesh to the commander laptop - GPS, text, and status intact.

Why This Backup Wins

Full GPS & UI

Phones keep their friendly UI and GPS hardware- satellites never go offline with the towers.

18 650-powered

Nodes sip <1 W; a single 3 000 mAh cell runs ~24 h. Plug any power bank for multi-day ops.

Mesh Range ×3

Each relay hop boosts coverage; hilltop node reaches 2–5 km LOS on 433 MHz in Israel’s licence-free band.

Open-Source Core

Powered by Meshtastic, audited by a global hobbyist & SAR community.

Sub ₪350 / Node

LILYGO T-Beam @ 433 MHz retails ~US$80 / ₪290; bulk buys cut that further.

12-48 h Bridge

Keeps data flowing until large-scale IDF comms arrive- then nodes become redundancy.
